Create highly responsive charts & graphs in WordPress with Power Charts using the advanced D3.js visualization library.

== Description ==

Power Charts allows you to create highly responsive, flexible charts & graphs for your WordPress site. Choose from an array of configurable chart types.

All charts and graphs are created using the powerful D3.js library.

== Installation ==

1. Via the WordPress admin go to Plugins => Add New.
2. Enter 'Power Charts' (without quotes) in the text box and click the 'Search Plugins' button.
3. In the list of relevant Plugins click the 'Install' link for Power Charts on the right hand side of the page.
4. Click the 'Install Now' button on the popup page.
5. Click 'Activate Plugin' to finish installation.
6. Under the main Power Charts menu click 'Add New Chart'.
7. Select your desired chart type and click 'Create Chart'.
7. Use the [pc id="123"] shortcode in a page, post, or text widget to display a chart. Replace '123' with the specific ID of the chart you want to display.
